MedSync

Aplicacao CLI para organizar horarios de medicamentos e consultar enderecos de entrega ou retirada usando a API publica ViaCEP.

Entrega intermediaria

A nova versao consome uma API REST externa, valida o fluxo com testes automatizados e documenta como executar o projeto.

Como executar

  1. Instale as dependencias com pip install -r requirements.txt.
  2. Rode a aplicacao com python src/main.py.
  3. Use a opcao de CEP para buscar um endereco no ViaCEP.

Qualidade

Os testes rodam com python -m pytest e o lint com python -m ruff check ..
